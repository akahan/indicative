category: validations
---

Ensures the length of a string or array is not is not less than
the expected length
 
[source, js]
----
const rules = {
  password: 'min:6'
}
 
// or
const rules = {
  password: [
    rule('min', 6)
  ]
}
----
For customized build, you can import this rule as follows.
[source, js]
----
import { min } from 'indicative/builds/validations'
----
