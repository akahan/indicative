title: trim
category: sanitizations
---

Remove whitespace from both sides of a given string.
 
[source, js]
----
const sanitizationRules = {
  username: 'trim'
}
 
// or
const sanitizationRules = {
  username: [
    rule('trim')
  ]
}
----
For customized build, you can import this rule as follows.
[source, js]
----
import { trim } from 'indicative/builds/validations'
----
